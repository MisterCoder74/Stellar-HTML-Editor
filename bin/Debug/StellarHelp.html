Stellar HTML Editor Help

Stellar HTML Editor is an HTML5 editor made in VB.NET for personal and semi-professional use. It is fully compatible with HTML5, HTML4, Bootstrap, CSS3, Javascript, Ajax, PHP and CGI to provide the user with the most updated technology and experience, keeping at the same time high compatibility with the most famous and used back-end features. Stellar HTML editor allows for the creation of web pages via code by using both hand-coding and templates/wizards; the editor is provided with a fully extensible system of HTML, Bootstrap, CSS, JScript, PHP and CGI templates and snippets that can be easily expanded by the user.

Stellar HTML Editor features all the most common and used functions in an HTML editor, including: a quick taglist, HTML wizards for the most common elements (empty page, tables, links, lists, i-frames, forms, etc), a fast-tagging toolbar, code indenting, tag stripping, and a fully extensible snippets/templates system. The pages created during code writing can be tested out to your favourite external browser for prompt feedback.

Stellar HTML Editor features a fully extensible snippets/templates system. Templates and snippets are loaded in the Template Browser and called by the top menu; all snippets can be previewed for quick reference, and inserted in the current edited page. Snippets are sometimes divided into parts, i.e. an HTML5 snippet can have a CSS3 counterpart indicated in the code, which must be included in the same document for correct visualization. Snippets and templates include the latest and most useful technologies and elements for the creation of responsive and flexible web pages that can be visualized in all media types.
